# 秒记账 v6.10.3 覆盖安装优化版

本版修改：
- 保持 `applicationId` 为 `com.miaojizhang.app`，确保新版和旧版被系统识别为同一个应用。
- `versionCode` 从 13 提升到 14，`versionName` 改为 `6.10.3`，满足 Android 升级安装要求。
- 新增固定签名文件 `miaojizhang-stable-signing.jks`，并配置 debug / release 都使用同一签名，避免 GitHub Actions 每次生成不同 debug 签名导致无法覆盖安装。
- GitHub Actions 改为同时构建 debug 与 release APK。
- 未加入“新版首次启动自动备份”逻辑，保持数据逻辑不变。

重要说明：
- 覆盖安装要同时满足：包名一致、签名一致、versionCode 递增。
- 如果手机上当前安装的旧包是之前用随机 debug 签名打出来的 APK，Android 仍可能提示签名冲突，无法直接覆盖。这个限制无法在新包里绕过；需要先备份数据，再卸载旧包安装本版。
- 从本版开始，只要后续继续保留这个 JKS 签名文件，并继续递增 versionCode，就可以正常替换安装且保留应用数据。

- 去掉原生覆盖启动页，避免挡住页面导致无法点击

- 保留预算首页、记账、记录修改/删除、分类、统计、我的、备份导出

修复：启动页跳转独立于业务 JS，避免卡在启动页。
